# Blockchain-based Medicine Registration & Transfer System 🏥💊

## Description 📜
This project is a blockchain-based system for registering, verifying, and transferring medicines. It enables various roles, including Admin, to manage medicine batches, add quality checks, approve medicines, and transfer them between parties. The system also supports generating QR codes for transactions and verifying transactions via smart contracts.

## Key Features ⚙️
- **Medicine Registration**: Register new medicine batches with relevant details. 📝
- **Medicine Verification**: Verify the details of a specific medicine batch using the batch number. 🔍
- **Transfer Medicine**: Transfer medicine from one party to another. 🔄
- **Approve Medicine**: Admin users can approve a medicine batch. ✅
- **Quality Check**: Add quality notes to the medicine batches. 🏷️
- **Transaction History**: View the complete transaction history of medicine transfers. 📜
- **QR Code Generation**: Generate QR codes to verify transactions. 📱

## Technologies Used 💻:
- **React.js**: Frontend framework for building the user interface. ⚛️
- **Web3.js**: Interact with the Ethereum blockchain. ⛓️
- **Solidity**: Smart contract for handling transactions. 💼
- **QRCode**: Generate QR codes for transactions. 📲

## How to Use 🚀

### Setting up the project:

1. **Clone the Repository**:
    ```bash
    git clone <repo_url>
    cd <project_folder>/frontend
    ```

2. **Install Dependencies**:
    ```bash
    npm install
    ```

3. **Start the Application**:
    ```bash
    npm start
    ```

    This will run the app on `http://localhost:3000`. 🌐
