📝 Step 3: Import Ganache Accounts

Important: After adding the network, you need to import the Ganache accounts into MetaMask.

  1. Open Ganache GUI or CLI
  2. Click on an account to see its private key
  3. In MetaMask: Account dropdown → Import Account
  4. Paste the private key and click Import
  5. Repeat for all 3 accounts (Admin, Manufacturer, Retailer)

✅ Step 4: Verify Connection

After importing accounts:

  • Switch MetaMask to "Ganache Local" network
  • Select the account matching your role (e.g., Admin account for admin login)
  • Go to login page and login!
